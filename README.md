# Institutional Equity Valuation Dashboard

## Overview

A Python-based institutional equity research platform that automates:

- Financial statement analysis
- Ratio analysis
- Discounted Cash Flow (DCF) valuation
- Comparable company valuation
- Interactive dashboards
- AI-assisted investment research

## Features

- Company overview
- Financial statements
- Financial ratios
- Growth analysis
- DCF valuation
- Comparable company analysis
- Sensitivity analysis
- Investment recommendation
- PDF report generation

## Technology Stack

- Python
- Pandas
- NumPy
- Streamlit
- Plotly
- yFinance
- GitHub Codespaces
